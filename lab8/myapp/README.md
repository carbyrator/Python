1) Целью лабораторной работы является создание веб-приложения с использованием модели MVC (Model-View-Controller), интеграцией с API для получения актуальных курсов валют и использованием шаблонизатора Jinja2 для рендеринга HTML-страниц. Также необходимо тестирование приложения на корректность работы моделей, контроллеров и функции получения валют.
2) Описание предметной области
Приложение включает следующие модели:
Модели:
Модели для пользователей и валют. Пользователи имеют свой уникальный ID, имя и данные о выбранных валютах, которые отображаются на странице.
Валюты представляют собой данные о курсах валют, получаемых через API.
Связи:
Пользователь может иметь несколько валют в своем профиле, а валюты имеют курс, который обновляется по API.

4) 1) Модели в проекте представляют данные, такие как пользователь, автор, валюта и связь между пользователем и валютой. Каждая модель использует инкапсуляцию, скрывая данные за закрытыми атрибутами и предоставляя доступ к ним через геттеры и сеттеры.
   2) Маршруты и обработка запросов реализуются с использованием фреймворка для веб-разработки (Flask). Маршруты сопоставляют URL-адреса с функциями, которые обрабатывают запросы от клиента.
   3) Шаблонизатор Jinja2 используется для генерации HTML-страниц на сервере. Он позволяет вставлять динамические данные в HTML-шаблоны, что упрощает создание веб-страниц.
Инициализация Jinja2 Environment:

from jinja2 import Environment, FileSystemLoader
# Устанавливаем путь к папке с шаблонами
env = Environment(loader=FileSystemLoader('templates'))
# Загружаем шаблон
template = env.get_template('index.html')
# Рендерим шаблон с данными
output = template.render(users=users)
   4) Функция get_currencies интегрирована в приложение для получения актуальных курсов валют через API. Она отправляет запрос к внешнему API, парсит ответ в формате JSON и возвращает курсы валют. Эти курсы затем используются для обновления данных в базе или отображения на веб-страницах.
